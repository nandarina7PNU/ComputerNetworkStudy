# 1.2 네트워크의 가장자리

인터넷은 크게 다음 두 영역으로 나누어 생각할 수 있다.

* **네트워크 가장자리(network edge)**: 컴퓨터, 스마트폰, 서버처럼 사용자가 직접 이용하며 응용 프로그램이 실행되는 부분
* **네트워크 코어(network core)**: 라우터와 스위치가 패킷을 목적지까지 전달하는 부분

이번 절에서는 네트워크 가장자리에 있는 장치들과 이 장치들이 인터넷에 연결되는 방식을 살펴본다.

---

## 1. 종단 시스템과 호스트

인터넷에 연결된 컴퓨터와 장치를 **종단 시스템(end system)**이라고 한다.

종단 시스템이라는 이름은 이러한 장치들이 인터넷의 중심부가 아니라 **네트워크의 가장자리**에 위치하기 때문에 붙여졌다.

종단 시스템의 예는 다음과 같다.

* 데스크톱 컴퓨터
* 노트북
* 스마트폰과 태블릿
* 웹 서버와 이메일 서버
* 인터넷에 연결된 가전제품, 센서, 자동차 등의 IoT 장치

종단 시스템은 **호스트(host)**라고도 부른다. 호스트는 웹 브라우저, 이메일 프로그램, 웹 서버와 같은 응용 프로그램을 실행하기 때문이다.

> **호스트(host) = 종단 시스템(end system)**

즉, 호스트와 종단 시스템은 같은 의미로 사용된다.

---

## 2. 클라이언트와 서버

호스트는 일반적으로 **클라이언트**와 **서버**로 구분된다.

### 클라이언트

사용자가 직접 이용하면서 서버에 데이터나 서비스를 요청하는 장치이다.

예:

* 데스크톱 컴퓨터
* 노트북
* 스마트폰
* 태블릿

### 서버

클라이언트의 요청을 처리하고 데이터나 서비스를 제공하는 컴퓨터이다.

서버는 다음과 같은 작업을 수행한다.

* 웹 페이지 저장 및 전송
* 검색 결과 제공
* 이메일 저장 및 중계
* 동영상 스트리밍
* 모바일 앱 콘텐츠 제공

오늘날 대부분의 서버는 개별적으로 설치되기보다는 대규모 **데이터 센터(data center)**에 모여 있다.
데이터센터에 대한 내용은 문서 마지막에서 확인할 수 있다.

---

# 1.2.1 접속 네트워크

## 1. 접속 네트워크란?

**접속 네트워크(access network)**는 종단 시스템을 인터넷 경로상의 첫 번째 라우터와 물리적으로 연결하는 네트워크이다.

이 첫 번째 라우터를 **에지 라우터(edge router)**라고 한다.

예를 들어 가정에서 노트북으로 인터넷을 사용할 때 다음과 같은 경로를 거칠 수 있다.

> 노트북 → Wi-Fi 공유기 → 모뎀 → 통신 사업자의 네트워크 → 에지 라우터 → 인터넷

접속 네트워크는 사용 환경에 따라 크게 다음과 같이 구분할 수 있다.

* 가정용 접속 네트워크
* 기업 및 대학의 접속 네트워크
* 광역 모바일 무선 접속 네트워크

---

# 2. 가정용 접속 네트워크

대표적인 가정용 인터넷 접속 방식에는 다음이 있다.

* DSL
* 케이블 인터넷
* FTTH
* 5G 고정형 무선

---

## 2.1 DSL

**DSL(Digital Subscriber Line)**은 기존의 전화선을 이용하여 인터넷에 접속하는 방식이다.

가정에서 유선 전화 서비스를 제공하는 전화 회사가 DSL 인터넷 서비스도 제공하므로, 이 경우 전화 회사가 ISP 역할도 수행한다.

### DSL의 기본 구조

> 가정의 DSL 모뎀 → 기존 전화선 → 전화국의 DSLAM → 인터넷

가정의 **DSL 모뎀**은 디지털 데이터를 전화선으로 전송할 수 있는 고주파 신호로 변환한다.

전화국의 중앙국인 **CO(Central Office)**에는 **DSLAM**이 설치되어 있다.

**DSLAM(Digital Subscriber Line Access Multiplexer)**은 여러 가정에서 들어온 신호를 모으고, 아날로그 신호를 다시 디지털 데이터로 변환하여 인터넷으로 전달한다.

하나의 DSLAM에는 수백 또는 수천 가구가 연결될 수 있다.

---

## 2.2 DSL의 주파수 분할

DSL은 하나의 전화선으로 인터넷 데이터와 전화 음성을 동시에 전송한다.

이를 위해 서로 다른 주파수 대역을 사용한다.

| 채널    |     주파수 대역 | 역할                |
| ----- | ---------: | ----------------- |
| 전화 채널 |     0~4kHz | 일반적인 양방향 전화 통화    |
| 상향 채널 |    4~50kHz | 가정에서 인터넷으로 데이터 전송 |
| 하향 채널 | 50kHz~1MHz | 인터넷에서 가정으로 데이터 전송 |

이처럼 하나의 통신 회선을 여러 주파수 대역으로 나누어 사용하는 방법을 **주파수 분할 다중화(FDM)**라고 한다.

가정에 설치된 **스플리터(splitter)**는 전화 신호와 인터넷 데이터 신호를 분리한다.

* 전화 신호는 전화기로 전달
* 데이터 신호는 DSL 모뎀으로 전달

따라서 전화 통화를 하면서 동시에 인터넷을 사용할 수 있다.

---

## 2.3 DSL의 비대칭 구조와 한계

DSL은 일반적으로 다운로드 속도가 업로드 속도보다 빠르다.

이처럼 하향 전송 속도와 상향 전송 속도가 다른 접속 방식을 **비대칭 접속(asymmetric access)**이라고 한다.

가정 사용자는 일반적으로 데이터를 업로드하는 것보다 웹 페이지, 동영상, 파일 등을 다운로드하는 경우가 많기 때문에 하향 속도를 더 높게 배정한다.

DSL의 실제 속도는 다음 요인의 영향을 받는다.

* 가입한 요금제
* 가정과 전화국 중앙국 사이의 거리
* 전화선의 굵기와 품질
* 전기적 간섭

특히 DSL은 가정과 중앙국 사이의 거리가 멀어질수록 속도가 감소한다. 따라서 중앙국에서 너무 멀리 떨어진 가정은 다른 인터넷 접속 방식을 사용해야 한다.

---

# 3. 케이블 인터넷

케이블 인터넷은 전화선 대신 기존의 **케이블 TV망**을 이용한다.

가정은 일반적으로 케이블 TV를 제공하는 회사로부터 인터넷 서비스도 함께 제공받는다.

### 기본 구조

> 케이블 헤드엔드 → 광섬유 → 지역 접속 지점 → 동축 케이블 → 가정

케이블 사업자의 중심 시설을 **헤드엔드(head end)**라고 한다.

헤드엔드에서 지역 접속 지점까지는 광섬유를 사용하고, 지역 접속 지점에서 각 가정까지는 동축 케이블을 사용한다.

광섬유와 동축 케이블을 함께 사용하기 때문에 이를 **HFC(Hybrid Fiber Coax)**라고 한다.

하나의 지역 접속 지점은 일반적으로 수백 가구에서 수천 가구를 지원한다.

---

## 3.1 케이블 모뎀과 CMTS

가정에서는 **케이블 모뎀(cable modem)**을 사용한다.

케이블 사업자의 헤드엔드에는 **CMTS(Cable Modem Termination System)**가 있다.

CMTS는 DSL의 DSLAM과 비슷한 역할을 한다.

* 여러 가정에서 전송된 신호를 수신
* 아날로그 신호를 디지털 형식으로 변환
* 데이터를 인터넷으로 전달

케이블 인터넷 역시 하향 채널과 상향 채널로 구분되며, 일반적으로 다운로드 속도가 업로드 속도보다 빠른 비대칭 구조를 사용한다.

---

## 3.2 공유 방송 매체

케이블 인터넷의 중요한 특징은 여러 가정이 하나의 네트워크를 공유한다는 것이다.

케이블 인터넷은 **공유 방송 매체(shared broadcast medium)**이다.

헤드엔드가 보낸 패킷은 같은 지역 네트워크에 연결된 모든 가정 방향으로 전달된다. 또한 각 가정에서 보낸 데이터도 공유된 상향 채널을 통해 헤드엔드로 전달된다.

따라서 같은 지역에서 여러 사용자가 동시에 대용량 파일이나 동영상을 다운로드하면 각 사용자의 실제 속도가 낮아질 수 있다.

반대로 사용자가 적거나, 사용자들이 서로 다른 시간에 웹 페이지를 요청하면 높은 속도를 이용할 수 있다.

상향 채널도 여러 사용자가 공유하므로, 여러 가정이 동시에 데이터를 보내 충돌하는 것을 막기 위한 **다중 접속 프로토콜**이 필요하다.

---

## DSL과 케이블 비교

| 구분       | DSL                | 케이블 인터넷           |
| -------- | ------------------ | ----------------- |
| 기존 인프라   | 전화선                | 케이블 TV망           |
| 사업자 측 장비 | DSLAM              | CMTS              |
| 가정 장비    | DSL 모뎀             | 케이블 모뎀            |
| 회선 구조    | 가정과 중앙국 사이의 전화선 사용 | 여러 가정이 지역 케이블망 공유 |
| 주요 속도 영향 | 중앙국까지의 거리와 회선 품질   | 같은 지역 사용자의 동시 사용량 |
| 공통점      | 일반적으로 비대칭 접속       | 일반적으로 비대칭 접속      |

---

# 4. FTTH

**FTTH(Fiber To The Home)**는 통신 사업자의 중앙국에서 각 가정까지 광섬유를 연결하는 방식이다.

광섬유는 전화선이나 동축 케이블보다 훨씬 높은 전송 속도를 제공할 수 있으며, 기가비트급 인터넷 접속이 가능하다.

---

## 4.1 직접 광섬유

가장 단순한 방식은 **직접 광섬유(direct fiber)**이다.

중앙국에서 각 가정으로 별도의 광섬유를 하나씩 설치한다.

성능은 좋지만, 가정마다 개별 광섬유가 필요하므로 설치 비용이 크다는 단점이 있다.

---

## 4.2 AON과 PON

하나의 광섬유를 여러 가정이 공유하고, 가정에 가까운 지점에서 회선을 나누는 방식도 있다.

대표적인 광 분배 구조는 다음 두 가지이다.

* **AON(Active Optical Network)**
* **PON(Passive Optical Network)**

AON은 전기적으로 작동하는 네트워크 장비를 사용하며, 본질적으로 스위치드 이더넷 방식과 유사하다.

PON은 수동형 스플리터를 사용하여 하나의 광섬유를 여러 가정에 나누어 제공한다.

---

## 4.3 PON의 구조

PON에서는 각 가정에 **ONT(Optical Network Terminator)**가 설치된다.

ONT는 가정에서 사용하는 광신호 종단 장치로, 가정의 공유기와 연결된다.

기본 구조는 다음과 같다.

> 가정의 ONT → 전용 광섬유 → 지역 스플리터 → 공유 광섬유 → 중앙국의 OLT → 인터넷

지역의 **스플리터(splitter)**는 여러 가정을 하나의 공유 광섬유에 연결한다.

중앙국의 **OLT(Optical Line Terminator)**는 광신호와 전기 신호를 서로 변환하고 통신 사업자의 라우터를 통해 인터넷에 연결한다.

OLT에서 스플리터로 전송된 패킷은 스플리터에서 여러 가정 방향으로 복제된다.

---

# 5. 5G 고정형 무선

**5G 고정형 무선(5G fixed wireless)**은 기지국과 가정의 모뎀 사이에서 데이터를 무선으로 전송하는 가정용 인터넷 접속 방식이다.

기존 방식처럼 중앙국에서 가정까지 전화선, 동축 케이블 또는 광섬유를 직접 설치할 필요가 없다.

5G 고정형 무선은 **빔포밍(beam-forming)** 기술을 사용하여 특정 가정 방향으로 무선 신호를 집중적으로 전송한다.

가정에서는 다음과 같은 구조로 인터넷을 사용한다.

> 5G 기지국 → 가정용 5G 모뎀 → Wi-Fi 공유기 → 가정의 장치

모뎀과 Wi-Fi 공유기가 하나의 장치로 결합되기도 한다.

장점은 케이블 설치 비용을 줄이면서도 비교적 빠른 인터넷을 제공할 수 있다는 것이다.

---

# 6. 기업 및 가정의 접속: 이더넷과 Wi-Fi

기업, 대학, 가정에서는 일반적으로 **LAN(Local Area Network)**을 이용해 종단 시스템을 에지 라우터에 연결한다.

대표적인 LAN 기술은 다음과 같다.

* 유선 방식: 이더넷
* 무선 방식: Wi-Fi

---

## 6.1 이더넷

**이더넷(Ethernet)**은 기업, 대학 및 가정에서 가장 널리 사용되는 유선 LAN 기술이다.

사용자의 컴퓨터나 서버는 꼬임쌍 구리선을 통해 **이더넷 스위치**에 연결된다.

> 종단 시스템 → 이더넷 스위치 → 라우터 → 인터넷

여러 이더넷 스위치가 서로 연결되어 하나의 더 큰 네트워크를 만들 수도 있다.

일반 사용자 장치는 보통 100Mbps에서 수 Gbps의 속도로 스위치에 연결되며, 서버는 더 높은 속도로 연결될 수 있다.

---

## 6.2 Wi-Fi

무선 LAN에서는 사용자가 **액세스 포인트(access point)**를 통해 패킷을 송수신한다.

액세스 포인트는 보통 유선 이더넷을 통해 기업이나 가정의 네트워크에 연결되어 있다.

Wi-Fi는 **IEEE 802.11** 표준을 기반으로 한다.

Wi-Fi 사용자는 일반적으로 액세스 포인트에서 수십 미터 이내에 있어야 한다.

Wi-Fi는 다음 장소에서 널리 사용된다.

* 가정
* 대학
* 회사
* 카페
* 공항
* 호텔
* 비행기

Wi-Fi 역시 여러 사용자가 하나의 무선 전송 매체를 공유하므로, 표시된 최대 속도와 각 사용자가 실제로 얻는 속도는 다를 수 있다.

---

## 6.3 일반적인 가정 네트워크

오늘날 가정 네트워크는 광대역 접속 기술과 Wi-Fi를 결합하여 구성된다.

일반적인 구조는 다음과 같다.

> ISP 회선 → 모뎀 또는 ONT → 가정용 라우터 → 유선 및 무선 장치

가정용 라우터는 다음 기능을 결합하여 제공하는 경우가 많다.

* 인터넷 연결
* 유선 이더넷 연결
* 무선 액세스 포인트
* 가정 내부 장치 간 데이터 전달

이를 통해 가족 구성원은 노트북이나 스마트폰을 들고 집 안을 이동하면서 계속 인터넷을 사용할 수 있다.

# 7. 광역 무선 접속

스마트폰과 태블릿은 셀룰러 이동통신망을 통해 인터넷에 연결된다.

대표적인 광역 무선 접속 기술은 다음과 같다.

* 3G
* LTE
* 4G
* 5G

모바일 장치는 이동통신 사업자가 운영하는 기지국과 무선으로 패킷을 주고받는다.

Wi-Fi는 일반적으로 수십 미터 범위에서 사용되지만, 셀룰러 네트워크는 기지국으로부터 수 킬로미터에서 수십 킬로미터 떨어진 곳에서도 사용할 수 있다.

| 구분      | Wi-Fi       | 셀룰러 네트워크        |
| ------- | ----------- | --------------- |
| 연결 장비   | 액세스 포인트     | 이동통신 기지국        |
| 일반적인 범위 | 수십 미터       | 수 킬로미터 이상       |
| 사용 장소   | 가정, 회사, 카페  | 이동 중인 넓은 지역     |
| 대표 기술   | IEEE 802.11 | 3G, LTE, 4G, 5G |

4G는 실제 환경에서 수십 Mbps의 다운로드 속도를 제공하며, 5G는 이보다 더 높은 속도와 낮은 지연시간을 목표로 한다.

# 사례 연구: 데이터 센터와 클라우드 컴퓨팅

Google, Microsoft, Amazon, Alibaba와 같은 기업은 수만 대에서 수십만 대의 호스트가 설치된 대규모 데이터 센터를 운영한다.

데이터 센터는 단순히 인터넷에 연결되어 있는 것이 아니라, 내부 서버들을 연결하는 복잡한 데이터 센터 네트워크도 포함한다.

데이터 센터는 현대 인터넷 애플리케이션이 실행되는 핵심 기반 시설이다.

---

## 1. 데이터 센터의 주요 목적

데이터 센터는 크게 세 가지 역할을 수행한다.

### ① 인터넷 콘텐츠 제공

사용자에게 다음과 같은 콘텐츠를 제공한다.

* 웹 페이지
* 상품 정보
* 검색 결과
* 동영상
* 이메일
* 모바일 앱 데이터

### ② 대규모 데이터 처리

기업 자체의 업무를 위한 대규모 계산과 데이터 처리를 수행한다.

수많은 서버를 동시에 이용하여 작업을 나누어 처리하므로 **대규모 병렬 컴퓨팅**과 **분산 컴퓨팅**이 가능하다.

### ③ 클라우드 컴퓨팅 제공

다른 기업이 자체 데이터 센터를 구축하지 않고도 서버, 저장장치, 네트워크 등의 컴퓨팅 자원을 사용할 수 있도록 한다.

예를 들어 Airbnb와 같은 기업은 자체 데이터 센터를 직접 운영하는 대신 Amazon의 **AWS(Amazon Web Services)**를 이용하여 웹 서비스를 실행할 수 있다.

---

## 2. 데이터 센터의 호스트

데이터 센터에서 실제 작업을 수행하는 주체는 수많은 호스트이다.

이 호스트들은 다음과 같은 작업을 수행한다.

* 웹 페이지와 동영상 제공
* 이메일과 문서 저장
* 데이터베이스 처리
* 대규모 분산 계산

데이터 센터의 호스트는 흔히 **블레이드(blade)** 형태로 제작된다.

블레이드는 피자 상자처럼 얇은 형태의 서버로, 일반적으로 다음 구성요소를 포함한다.

* CPU
* 메모리
* 디스크 또는 저장장치
* 네트워크 인터페이스

여러 블레이드는 **랙(rack)**에 수직으로 쌓인다. 하나의 랙에는 일반적으로 20개에서 40개의 블레이드가 들어갈 수 있다.

이러한 랙들은 고속 네트워크 장비를 통해 서로 연결되며, 전체적으로 하나의 거대한 데이터 센터 네트워크를 구성한다.

---

# 핵심 정리

* 네트워크 가장자리에는 컴퓨터, 스마트폰, 서버 등의 종단 시스템이 위치한다.
* 종단 시스템은 응용 프로그램을 실행하므로 호스트라고도 부른다.
* 호스트는 일반적으로 클라이언트와 서버로 구분된다.
* 접속 네트워크는 종단 시스템을 첫 번째 라우터인 에지 라우터에 연결한다.
* DSL은 기존 전화선을 사용하며, 데이터와 전화 신호를 서로 다른 주파수로 나누어 전송한다.
* 케이블 인터넷은 광섬유와 동축 케이블을 함께 사용하며, 여러 가정이 전송 매체를 공유한다.
* FTTH는 가정까지 광섬유를 연결하며, 높은 전송 속도를 제공한다.
* 5G 고정형 무선은 기지국과 가정용 모뎀 사이를 무선으로 연결한다.
* 기업과 가정에서는 이더넷과 Wi-Fi를 이용해 LAN을 구성한다.
* 셀룰러 네트워크는 Wi-Fi보다 훨씬 넓은 범위에서 모바일 인터넷 접속을 제공한다.
* 데이터 센터는 대규모 서버들을 이용해 콘텐츠 제공, 데이터 처리, 클라우드 컴퓨팅 서비스를 수행한다.
